Case Study: Paisool.com

Offerings: AI/ML

Company Profile

  • Startup with ~15 employees and a disruptor in the FMCG retail segment
  • Builds In house intelligent PoS devices that maintain consumer information and facilitate global access to user proﬁles

Problem

  • For this startup that connects shoppers with a retail store, it needed a way to connect individual PoS devices in a peer to peer fashion without a central cloud in between
  • The system is expected to be able to work with poor and slow connections and handle issues like ﬁrewalls etc

Solution

We developed a library for the webRTC data channel protocol and wrapped it in python for easy use. This was deployed on all the machines and scaled up to handle the number of connections they needed

Result

Our solution was able to handle up-to 50 connections per node simultaneously sending and receiving trafﬁc. The packaging enabled easy software updates without disruption

Technologies

Python (FFI), C/C++, Flask, REST
