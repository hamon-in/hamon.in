Case Study: QuackQuack

Offerings: AI/ML

Company Profile

  • Leading Indian dating app with 4M users
  • 1000+ new users per day and growing
  • Promises verified profiles of users, Interest based matchmaking, and fake & spammer free app

Problem

  • Manual verification of user profiles (5 employees, 35k/month)
  • Error prone, tedious and boring
  • Classifying profile information into vague, spam, informercial, genuine

Solution

Hamon developed machine learning techniques to create models that can help detect bad actors from genuine users and improve the usability of the app. After having a bad experience working with a vendor, Hamon was able to come in and quickly develop the right machines learning models and help QuackQuack

Result

82% Accuracy rate in detecting spams based on available models and timeframe of the project. Able to take off human curators

Technologies

SciKit Learn, Pandas, Jupyter Notebook, AWS EC2, Python webstack (Flask, PostgreSQL, SQL Alchemy)
