Case Study: Renesas Electronics

Offerings: AI/ML

Company Profile

  • Renesas Electronics, headquartered in Japan is a multi-billion dollar semiconductor solution provider. The company has over 30,000 employees

Problem

Renesas needed an IoT device management solution for the Synergy series of MCU’s. Being a hardware and embedded software company, the organization lacked expertise in modern cloud technologies and integration with embedded software

Solution

Developed a library that could be linked with any embedded application which would provided device management functionality Provided cloud backend to communicate and manage devices remotely

Result

Initial proof of concept demonstrated OTA and monitoring capabilities successfully

Technologies

Embedded development software, C, Amazon AWS, REST API’s, Flask
